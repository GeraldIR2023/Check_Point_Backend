### Tech Stack

- **Framework:** NestJS (Node.js)
- **Language:** TypeScript
- **Database:** PostgreSQL
- **ORM:** TypeORM / Sequelize (especifica el que usaste)
- **Tools:** Postman for API testing, Git for version control.

### Key Technical Features

- **Modular Architecture:** Designed with specialized modules for Products, Users, Orders, and Auth to maintain a clean separation of concerns.
- **Complex Business Logic:** Engineered a custom Discount Coupon Engine and a dynamic Shopping Cart with server-side validation.
- **Data Integrity:** Implemented relational database modeling with PostgreSQL, using DTOs (Data Transfer Objects) and Class Validator for robust input sanitization.
- **RESTful Standards:** Developed secure endpoints with structured JSON responses and appropriate HTTP status codes.

# How to Run

- Clone the repository.
- Install dependencies: npm install.
- Configure your .env file (Database credentials).
- Run in development mode: npm run start:dev.
